Machine Learning for Product Color Recommendation

This is an implementation of a product color recommendation system using machine learning.

The system will generate a random color set for the shoe. It will keep learning from how do you rate that, and will predict new recommended color combinations in the Recommendations section.

The more you train the system with rating buttons, the better it gets at recommending. Usually, after you train it for 30-40 times, you'll notice that it is recommending colors that you may actually like.
